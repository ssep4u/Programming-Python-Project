환경설정(이미지 파일을 불러올 수 있도록 환경설정을 먼저 해준다) :
    1. cmd 창을 열어준다.
    2. pip install opencv-python 이라고 입력한다.

1. main.py 파일을 열고 Run 하여 실행한다.
나라 검색:
    2. 키보드로 1을 입력하여 나라를 검색한다.
    3. 나라 이름을 키보드로부터 입력한다. (ex: 미국)
    4. 입력한 나라의 관광지 설명이 출력되고 그곳에 관한 사진이 뜨게된다.
    5. 뜬 창을 클릭하여 아무키나 누르면 사진이 차례대로 뜨게 된다. (사진이 뜨는 순서는 관광지 설명순서와 일치하다.)
나라 음식 검색:
    6. 키보드로 2를 입력하여 나라 음식을 검색한다.
    7. 나라 이름을 키보드로부터 입력한다. (ex: 미국)
나라 추가:
    8. 키보드로 3을 입력하여 나라를 추가한다.
    9. 추가할 나라 이름을 입력해준다.
음식 메뉴 추가:
    10. 키보드로 4를 입력하여 음식 메뉴를 추가한다.
    11. 추가할 음식 이름을 입력해준다.
종료:
    12. 키보드로 5를 입력하여 모든창을 종료한다.
