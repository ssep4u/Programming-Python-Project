##✔ 실행방법
#####1. 폴더를 압출 헤제 후 main.py를 run 한다.
#####2. 로그인이나 회원가입을 선택한다.
###### ---> 회원가입 시 가입 후 로그인
#####3. 메뉴를 선택한다.
###### ---> 로그인 및 로그아웃
###### ---> 게시물 둘러보기
###### ---> 동물 종류별 검색
###### ---> 입양신청
###### ---> 내 정보 확인
###### ---> 분양 게시물 등록
###### ---> 종료
#####4. 선택한 메뉴대로 실행한다.
#####5. 다 끝났다면 종료메뉴를 선택해 프로그램을 종료한다.
