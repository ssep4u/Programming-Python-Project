처음 실행파일이름 : main.py
--------------------------------------------------
실행방법
1. 강아지 정보추가
2. 강아지 정보확인을 위해 입력 : 추가한 강아지 이름 입력
      - 있으면 정보출력
      - 없으면 정보 추가 가능
3. 머리와 몸 중 아픈 부위가 속해 있는 것을 선택
4. 머리를 선택했다면 
      - 1. 눈
      - 2. 입
      - 3. 귀
      - 세가지 부위 중 하나 선택
   몸을 선택했다면
      - 1. 다리
      - 2. 배
      - 세가지 부위 중 하나 선택
5. 선택한 부위에 나타나는 증상이 나옴
6. 강아지한테 있는 증상 선택
7. 결과 출력
8. 마지막으로 5번 입력
      - 재시작을 하고 싶다면 y
      - 그만 하고 싶다면 n
